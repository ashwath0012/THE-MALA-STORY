# 🌸 The Mala Story

A modern and elegant website for showcasing handcrafted bouquets, varmalas, and floral arrangements.

## ✨ Features

- Beautiful and responsive design
- Bouquet & Varmala collection showcase
- WhatsApp enquiry integration
- Instagram integration
- Mobile-friendly layout
- Smooth user experience

## 🛠️ Built With

- React
- Vite
- JavaScript
- CSS

## 🌐 Live Website

https://the-mala-story.vercel.app

## 📸 Project Preview

The Mala Story is designed to help customers explore floral collections and contact directly through WhatsApp for enquiries and bookings.
